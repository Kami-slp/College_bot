# Чат-бот для учебной части

Flutter приложение для автоматической обработки отчетов учебной части из Excel файлов.

## Возможности

Приложение поддерживает 6 типов отчетов:

1. **Отчет по расписанию** - подсчитывает количество пар по каждой дисциплине
2. **Отчет по темам занятия** - проверяет соответствие тем формату "Урок № _. Тема: _"
3. **Отчет по студентам** - находит студентов с проблемными оценками (ДЗ = 1, классная работа < 3)
4. **Отчет по посещаемости** - находит преподавателей с посещаемостью ниже 40%
5. **Отчет по проверенным ДЗ** - находит преподавателей с процентом проверки ниже 70%
6. **Отчет по сданным ДЗ** - находит студентов с процентом выполнения ниже 70%

## Установка

1. Убедитесь, что у вас установлен Flutter SDK
2. Установите зависимости:
```bash
flutter pub get
```

## Запуск

```bash
flutter run
```

## Использование

1. Запустите приложение
2. Выберите тип отчета из списка
3. Нажмите "Выбрать Excel файл" и выберите файл (.xls или .xlsx)
4. Дождитесь обработки файла
5. Просмотрите результаты отчета

## Структура проекта

- `lib/models/` - модели данных для различных типов отчетов
- `lib/services/` - сервисы для парсинга Excel и обработки отчетов
- `lib/screens/` - экраны приложения (главный экран и экран отчета)

## Зависимости

- `excel` - для работы с Excel файлами
- `file_picker` - для выбора файлов
- `data_table_2` - для отображения таблиц данных (зарезервировано для будущих улучшений)
